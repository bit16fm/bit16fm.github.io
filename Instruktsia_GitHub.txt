1. Скачиваем git с оф. сайта 
https://git-scm.com/download/win
Скачиваение произойдет автоматически

2. Устанавливаем git

3. Регистрируемся на сайте github 
https://github.com/
И создаем первый репозиторий вида ваш_логин.github.io

4. Создаем папку с сайтом, которую будем синхронизировать с github

5. Из только что созданной папки запускаем командную строку и пишем команду
git init

6. Переносим файлы нашего сайта в папку с git 

7. В командной строке пишем команду 
git add .
(Точка в конце обязательна)

8. Проверяем что файлы добавлены под контроль версий командой:
git status 

9. Далее коммитим наш сайт в созданный на GitHub репозиторий, используя следующие команды (естественно нужно заменить "ваш_логин" на тот логин, который вы указывали при регистрации на github):

git commit -m "second commit"
git remote add origin https://github.com/bit16fm/bit16fm.github.io.git
git push -u origin master

После последней команды вас попросят ввести в терминал логин и пароль, нужно ввести данные для входа на GitHub.

Теперь сайт доступен по ссылке ваш_логин.github.io

команда на отправку  
git add . 
git commit -m "маслихат загрузил"
git push -u origin master
